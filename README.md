# Operating-System-Interface-In-Python
Η εργασία επιδεικνύει, μέσω εκτενώς σχολιασμένων παραδειγμάτων κώδικα, τις ιδανικές πρακτικές για τη διασφάλιση της φορητότητας (portability) του κώδικα μεταξύ διαφορετικών λειτουργικών συστημάτων (π.χ. Windows, macOS, Linux). 

Η βιβλιοθήκη os της Python αποτελεί ένα θεμελιώδες εργαλείο για κάθε προγραμματιστή. Παρέχει την απαραίτητη αφαίρεση (abstraction) για την εκτέλεση λειτουργιών συστήματος αρχείων με τρόπο που παραμένει ανεξάρτητος από την πλατφόρμα.
Μέσα από την παρούσα εργασία, αναλύσαμε μέσω σχολιασμένου κώδικα τις βασικές λειτουργίες που καλύπτουν τον κύκλο ζωής της διαχείρισης αρχείων:
1. Πλοήγηση: Κατανόηση του CWD (os.getcwd, os.chdir).
2. Ασφαλής Σύνθεση Διαδρομών: Η κρίσιμη σημασία του os.path.join().
3. Επισκόπηση: Έλεγχος ύπαρξης (os.path.exists), τύπου (.isfile, .isdir) και περιεχομένων (os.listdir).
4. Τροποποίηση: Δημιουργία (os.mkdir, os.makedirs), μετονομασία/μετακίνηση (os.rename) και διαγραφή (os.remove, os.rmdir).
5. Προχωρημένες Λειτουργίες: Η ανάγκη για τη βιβλιοθήκη shutil (ιδίως shutil.rmtree) για αναδρομικές και σύνθετες λειτουργίες.
6. Ανθεκτικότητα: Η χρήση try...except...finally για τον χειρισμό σφαλμάτων και τη διασφάλιση καθαρισμού.
